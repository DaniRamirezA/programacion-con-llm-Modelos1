## 🧠 Contenido

Se desarrollaron cuatro problemas:

* 🐼 **2 problemas con pandas**

  * Transformación de datos
  * Agrupación y análisis por grupos

* 🤖 **2 problemas con sklearn**

  * Clustering (KMeans + silhouette score)
  * Reducción de dimensionalidad (PCA)

Cada problema incluye:

* Enunciado claro (`.txt`)
* Función a implementar
* Generador de casos de uso (`.py`) con:

  * Datos aleatorios
  * Input estructurado
  * Output esperado (ground truth)
